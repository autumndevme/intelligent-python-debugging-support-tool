# Intelligent Python Debugging Support Tool

An intelligent Python debugging support tool designed to help novice programmers better understand Python errors through simplified explanations.

## Overview

Traditional compiler errors can be confusing for beginner programmers and may contribute to frustration, low confidence, and disengagement.

This project addresses that challenge by translating technical Python errors into beginner-friendly explanations while tracking student debugging behavior for educational research purposes.

## Features

- Detects beginner and intermediate Python errors
- Provides simplified explanations
- Suggests possible fixes
- Tracks user behavior
- Logs debugging attempts
- Measures time-to-resolution
- Supports both terminal and web interfaces
